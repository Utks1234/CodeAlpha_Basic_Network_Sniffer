# Basic Network Sniffer

## Overview
This project is a simple Network Sniffer built using Python and the Scapy library. It captures network packets in real time and displays useful information such as source IP address, destination IP address, protocol type, ports, and packet payload details.

The project helps understand how data travels across networks and provides practical exposure to network protocols such as TCP, UDP, and ICMP.

---

## Features

- Capture live network packets
- Display Source and Destination IP Addresses
- Identify Protocol Types (TCP, UDP, ICMP)
- Display Source and Destination Ports
- Show Packet Size and Payload Information
- Real-time Packet Monitoring
- Lightweight and Easy to Use

---

## Technologies Used

- Python 3.x
- Scapy Library
